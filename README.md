Composer 是PHP的类包依赖管理工具，用它可以轻松的引用第三方类包，
类似于node的npm、ruby的bundler、CentOS中的yum等

Composer为我们解决了两个大问题：

1）类库的依赖管理（尤其是N多类库的应用系统）
2）类的自动加载机制（Symfony/Zend/Laravel等均引用了Composer的类加载器）
